## What is it?
This is GIS based on Saint Petersburg

## How to install
1. Install XCode
2. Download repository archive
3. Download ArcGIS framework [iOS SDK](https://developers.arcgis.com/ios/install-and-set-up/)
4. Add ArcGIS framework to the project
